# 🧀 Say Cheese!

This browser extension allows the user to instantly hear the pronunciation of any word or sentence on a website.

---

## Installation:
### The Current version is only tested on Windows!
### 1. Adding it to browser:
* Download the project files to a folder on your computer.
* Open your browser and navigate to the Extensions page:
   * `brave://extensions` (Brave)
   * `chrome://extensions` (Chrome)
   * `edge://extensions` (Microsoft Edge)
* Turn on Developer mode (usually found in the top right corner)
* Click the Load unpacked button.
* Select the folder containing the extension files.

### 2. Adding languages:
This extension can only use the voices downloaded on Windows. 
If you select a language that's not downloaded it will use the default voice.
* To download new languages go to your Windows Setting and look for Speech.
* Under Manage voices you can add new voice languages.
* Upon installing a new voice it's reccommended to close and reopen your browser.

---

## How to use:
* Highlight any word or sectence on a website and right-click on it.
* Select Say Cheese and choose a language
